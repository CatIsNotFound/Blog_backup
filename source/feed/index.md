## 概述

本站现已支持使用 RSS 订阅。

RSS 订阅是一种基于 XML 的协议，只要将 RSS 订阅地址添加/导入到 RSS 阅读器中，就可以定期收到文章更新的通知。O(∩_∩)O

本站的 RSS 订阅地址如下：

```
https://catisnotfound.github.io/atom.xml
```

```
https://catisnotfound.github.io/rss.xml
```

## RSS 阅读器

目前你可以通过电脑端或手机端下载对应的 RSS 阅读器，然后将本站的 RSS 订阅地址添加到阅读器中即可。
